# TAMIMOS -> Linux Systems Build Plan

Each week is one Linux systems project that reinforces OS concepts.

## Week 1: Custom Linux Shell
Build a custom Unix shell in C with parsing, fork/exec, built-ins, pipes, and redirection.

## Week 2: Memory Allocator
Implement malloc/free/calloc/realloc from scratch with a free-list strategy and coalescing.

## Week 3: Thread Library and Synchronization
Build user-level thread APIs, mutexes, condition variables, producer-consumer queue, and concurrent hash map.

## Week 4: FUSE File System
Build a userspace file system with inodes, directory entries, and free block bitmap on a virtual disk file.

## Week 5: Kernel Module plus System Call
Write a loadable module with /proc output, then add and call a custom Linux syscall in a rebuilt kernel.

## Final Outcome
By the end, this becomes a complete Linux systems portfolio that mirrors core OS topics through real implementation.
